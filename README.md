# service-demo-pwa
A dynamic, multi-tenant Progressive Web Application (PWA) engineered for service-based industries.

Built with React, this prototype utilizes a unique "Chameleon Architecture" driven by an Agency Toolbar. It demonstrates advanced state management by dynamically injecting CSS variables and updating routing, booking logic, and service menus in real-time to adapt to multiple industry profiles (Salon, Auto Mechanic, Property Maintenance) from a single unified codebase.
